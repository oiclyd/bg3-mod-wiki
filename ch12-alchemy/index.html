    炼金攻略¶

    塔莎炼金术¶

    Tip

    出处模组：装备-骨铁饰品-EldertideArmament

    关键物品：塔莎的炼金术大全（书籍）、巫后护身符（护符）

    获取方式：炼金术大全在教程宝箱和部分商人处购买，巫后护身符建议用MCM简易作弊获得

    合成方式：通过巫后护身符给予的二环法术召唤合成台，投入对应炼药素材获得药剂

    纷争炼金术¶

    合成技巧：纷争炼金术炼制时只会消耗合成台右侧的物品，把白金之尘放在坩埚左侧即可无限使用同一份白金之尘。

    Tip

    出处：纷争系列模组

    关键物品：炼金日记（书籍）、炼金坩埚（物品）

    核心炼制过程 - 白金之尘¶

    1. 秋水仙聚合物 + 颠茄聚合物 = 黑土
    2. 黑土 + 独角兽角聚合物 = 白垩
    3. 白垩 + 麒麟丝聚合物 = 黄金
    4. 黄金 + 匕根草聚合物 = 白金之尘
