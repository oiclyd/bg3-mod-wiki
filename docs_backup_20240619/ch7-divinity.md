# 专长攻略

## 封神之路

共28个专长，通过杀敌获取灵魂 -> 封神之基 -> 神力等级的进阶体系。

### 专有名词

- **一人份灵魂**：10+2*max(力量调整值, 魅力调整值)层
- **灵魂献祭**：战斗外对1HP活物释放，获取30人份灵魂，每长休一次
- **封神之基**：每5000灵魂=1层，最多16层。每层：全属性+2、攻击+1、AC+1、先攻+1、移动+3m、跳跃+1.5m、生命+20

## 泰坦之握

- 角色本身（不含装备属性）力量属性18点后才可点此专长。
- 想双持巨武器：右键"装备"一把到主手，双击另一把进副手（顺序可能反过来）。
- 装备后武器栏只显示一把巨武器是正常的！主要看角色界面是否手持两把。
- 若两件巨武器同名会叠在主手，别拿同名巨武器。
- 读档可能刷新部分武器属性，真出怪事就把武器扔给队友再拿回来。
- 战斗中被打落武器可能装不回去，小心缴械！
- 一手盾牌一手巨武器：先装盾再装巨武器，摘盾比较麻烦。

## 鲜血恩惠

排序必须在所有法术上面，否则某些东西没有特效。

### 专长

- **狂人之路**：使用临时武器投掷时，伤害附带(2d8*力量调整值)+目标最大生命值1%
- **契约启迪**：邪术师专长
- **法师秘术**（仅法师可点，其他职业点了没用）：
- 等级4解锁：秘法锁、火球术、敲击术、死者交谈
- 等级7解锁：闪电束、造水术、魔法飞弹
- 等级10解锁：高等法师护甲、双重法理、秘法之门
- 等级14解锁：连锁闪电、寒冰锥、伪装术
- 等级17解锁：高等隐形术、念动力、秘法庇护
- 等级20解锁：鬼火、寒风天衣、完美之风
- 除高等隐形术与死者交谈外全部有增强，例如火球术每回合免费释放一次不消耗动作点和环位
- **牧师秘术**：治疗时额外恢复环位*3d8
