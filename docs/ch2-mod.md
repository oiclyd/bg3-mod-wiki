# MOD工具与教程

### MOD增删改说明


### 核心原则：关闭游戏操作


1. 无论是添加/删除/更新mod，都必须关闭游戏。
2. 无论删除什么mod都得还原游戏原始数据。
3. 部分mod更新也是同理。
4. 部分mod游戏中途删除会坏档。
5. 每次新增/删除/更新mod都要记得，点击[保存]+[导出]。


### 标准步骤

游戏中还原数据（例如洗点/扔掉装备/切换难度） -> 保存游戏 -> 退出游戏 -> 管理器删除/更新指定mod -> 管理器保存+导出 -> 进入游戏读档


### 具体操作示例


#### 删除职业

游戏中洗点成其他职业，然后按照步骤操作


#### 删除装备/道具

游戏中销毁（扔掉指定装备/道具 or 召唤教程宝箱 -> 扔进去让他随时间消失）。一定要确认身上和背包没有你要删除的mod道具，然后按照步骤操作。


#### 删除法术/专长

游戏中所有可控角色洗点成无施法职业，并确认所有队友没有点你要删除的专长，然后按照步骤操作。


### 特殊说明

- 添加mod一般没有影响，除非冲突，本包无冲突，外部下载的需自行排查
- 职业mod必须安装职业框架！
- 卸载Mod：删除MOD文件夹即可，其他文件对游戏无影响

#### MCM装备幻化说明


- 请使用 DX11 启动游戏，Vulkan可能不稳定或崩溃
- 创建"预设"：为整个队伍管理完整的套装，包括护甲、服装、染色、武器外观等
- 即时切换预设：在不同预设之间切换时，整个队伍的外观会立即更新
- 保留原始属性：装备在采用预设外观的同时，会保留其原始属性
- 跨槽位外观：允许在不同槽位显示物品（例如，在靴子槽显示手套）
- 附加装饰性视觉效果（VFX）：为装备和角色附加装饰性视觉效果（火焰武器、发光效果等），对游戏玩法没有影响
- 即时预览物品和染色：可以即时预览物品和染色，保存收藏夹，并隐藏不想显示的外观
- 完全支持多人游戏：联机时请保证所有人的MCM设置都一致
- 可分享的预设：可以导出为文件，发送给朋友，或者打包到你自己的模组中
- 如果使用过幻化水晶：请在MCM创建外观套装之前，先解除已装备物品的幻化


#### 幻化水晶说明


- 幻化水晶是Vulkan模式下的替代品
- 请不要使用幻化水晶去幻化武器！

---

> **警告：** 切勿跨体型或跨种族使用预设！请先通过"换种族技能"更改至预设的种族及体型（整合包内[技能-重新捏脸-AppearanceEditEnhanced]的功能）


#### 使用步骤


1. 务必先存档！
2. 载入游戏
3. 按住 `Shift + E` 打开窗口，**对着角色模型按鼠标中键**（关键）
4. 选择预设并应用
5. 点击"打开角色编辑器刷新头像"以更新角色


#### 注意事项


- 应用预设后，侧边栏角色头像有时会消失；再招募一名队友即可恢复。
- 角色编辑界面的"确认"按钮可能失灵（可先切换任意头部再切回，即可正常确认），该Bug仍在修复中。
- 若出现部位重叠，重新应用一次预设即可解决。


#### 导出预设须知


- 给预设命名，若分享请附上详细说明（例：`HUM_M_T2`）
- 如使用Mod，注明Mod名称或链接
- 导出后不会立刻显示，需点击"扫描预设文件"刷新列表
- 导出路径：`C:\Users\<你的用户名>\AppData\Local\Larian Studios\Baldur's Gate 3\Script Extender\CC_Preset`
- 从别处获取的预设放入上述目录并刷新列表，文件名按顺序：`P_1.json`、`P_2.json`...

#### A. 移除威尔的恶魔形态，恢复原来外观


```
Osi.RemoveTransforms("S_Player_Wyll_c774d764-4a17-48dc-b470-32ace9ce447d")
```


效果：会把威尔的"恶魔变身外观"移除掉；威尔会恢复到变身前的外观和头像；如果你用捏脸/自定义改过威尔，这里会恢复到你自定义后的样子。


#### B. 移除影心的白发新发型，恢复老样子


以下三行都要输入并回车，缺一行都可能不会生效：


```
Osi.RemoveCustomVisualOvirride("S_Player_ShadowHeart_3ed74f06-3c60-42dc-83f6-f034cb47c679", "e05b0f79-d3af-41eb-b0b2-1164b6f0debf")
Osi.RemoveCustomVisualOvirride("S_Player_ShadowHeart_3ed74f06-3c60-42dc-83f6-f034cb47c679", "c7f78eb7-6858-4f02-836b-d8c8b56c34fc")
Osi.RemoveCustomMaterialOverride("S_Player_ShadowHeart_3ed74f06-3c60-42dc-83f6-f034cb47c679", "831f8c8a-4101-4265-ad55-1a57217d2af7")
```


效果：移除影心的"新白色斜刘海发型"和那一套染发材质；让影心回到原本的黑发老发型（或你之前自定义的发色/发型）。


#### SE控制台使用方法


1. **什么是SE控制台？** 当你用Script Extender（脚本拓展器）启动博德之门3时，会自动弹出一个黑色的命令窗口，名为"BG3 Script Extender Debug Console"，这个黑色窗口就叫做SE控制台。
2. **怎么切到SE控制台？** 保持游戏在运行，不用退出到主菜单。把控制台切到最前面，用鼠标点一下黑色窗口内部，确保键盘输入焦点在这个窗口里。
3. **怎么输入命令？** 先按一次Enter（回车键），让光标跑到新的一行。把命令一行一行复制粘贴进去，每粘贴一行按一次Enter执行。执行成功不会有明显提示，只会在窗口里多出几行日志文字，这是正常的。
