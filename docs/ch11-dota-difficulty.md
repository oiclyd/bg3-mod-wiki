# Dota难度

#### 名词解释

- **抗性：**常规怪能拥有的50%减伤的抗性，能用易伤状态抵消
- **免疫：**免疫特定状态或伤害，减伤100%，需要特定buff或专长破除
- **减伤：**高难特有特性，直接减少绝大多数伤害数值
- **真伤：**无视抗性、免疫和减伤的特殊伤害

#### 高难思路

!!! warning "注意"
    高难构筑与原版完全不同，dota、谪仙、封神三个难度都有减伤。

- 武僧等多段附伤职业几乎沦为地板砖
- 需要保证伤害构成纯粹，不超过三个伤害类型
- 后期敌人人均全免疫，造易伤手段失去意义
- B类附伤和C类附伤效果远高于A类
- 需要堆高先攻一回合清场

#### Dota难度开启建议

!!! danger "警告"
    前期不要勾选boss，难度不是一个数量级！

- 建议在二级点出购物商店的专长
- 在林地戴摩手上买到自带法术反制和法术无效化结界的法杖之前不要开启

#### Dota专长隐藏属性加成

| 属性 | 加成效果 |
| --- | --- |
| 力量+1 | 生命+5，近战攻击+1 |
| 敏捷+1 | 远程攻击+1，攻速+1 |
| 智力+1 | 法术攻击+2 |
| 魅力+1 | 获取dota金币基础值增加 |

**攻速说明：**每有一点攻速，有1%概率增加一个标动，一回合只能生效一次，多段攻击会判定多次。

#### 出装建议

!!! warning "通用建议"
    所有队友攒150金币买可靠铁铲（最稳定的金币来源）。每一章限定增加两件成品装备（房规推荐）。点金手能更快赚钱，但来钱太快不太建议出。
      物理职业出装
- 第一件 鱼叉（拿四打是最高优先级）
- 第二件 百分比伤害装备或绝刃（拿无视免疫抗性）
法系职业出装
- 第一件 阿哈利姆神杖
- 第二件 玲珑心（可与散件增加选择目标的效果叠加）
辅助出装
- 第一件 否决
- 第二件 永恒之盘

#### 敌人机制应对

| 机制 | 说明 | 应对方法 |
| --- | --- | --- |
| 庇护术 | 无法被选定为攻击目标 | 用AOE技能攻击，推荐游侠猎杀者的无副动荆苞术，6环法术2e闪电链可无视庇护直接攻击 |
| 反伤 | 通常是近战反伤 | 先手开出无敌（黑皇杖或法术无效化结界） |
| 投射物反射 | 反射投射物攻击 | 套无敌再打，或让近战砍，或使用非投射物技能 |
| 复活/无敌 | 敌人复活或进入无敌状态 | 否决移除无敌、统御头盔直接强控、慧光连击移除无敌状态、用法术无效化结界的无敌拖时间、谪仙恶鬼道直接移除复活 |
| 0血怪 | 血条为0但活着，通常是bug | 继续攻击突破减伤，或读档/开挂 |
| 高假血怪 | 假血不计入总血量 | 当作血条特别厚的怪打 |
| 超高血条敌人 | 超过100万血 | 百分比伤害上限约46000，应对：多重百分比伤害，或用特殊装备/专长突破伤害上限 |
