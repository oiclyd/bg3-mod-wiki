### 职业说明

!!! tip
    **灭法之影**，包含20级内容。本MOD包含主职+专长。

#### 1. 真实属性

!!! tip
    开卡全属性-10（负调整值所带来的减益已修正），开卡时需要加力量不然无法更换装备！

#### 2. 负魅力

!!! tip
    负魅力为职业设定，与长相无关。杀敌数量过多导致魅力下降，从而获得物理说服等。

#### 3. 轮回乐园系统

!!! tip
    自带轮回乐园系统，不需要学习专长即可获取（其他职业需通过学习专长获取）。

#### 4. 猎杀者定位

!!! tip
    灭法之影为**'猎杀者'**，猎杀者是一些天赋特殊的契约者，这些契约者的天赋更加适合战斗。此外'乐园'还会发布猎杀任务，清理'违规'的契约者，从而获得额外的奖励。**暂未实装**

#### 5. 乐园商店

!!! tip
    灭法之影拥有独特的乐园商店（如夏之季餐厅、荣誉商店等）。

#### 6. 升级宗师被动

!!! tip
    达到一定等级后，需要'乐园币'和不同种类的**'灵魂结晶'**升级赋予终极能力。技法核心能力提升需要**'原初碎片'**，只能通过击杀特殊敌人获取，升级后获得巨额的提升。

#### 7. 属性值奖励

!!! tip
    达到目标等级和属性后可获得特殊被动。

#### 8. 断魂影天赋

!!! tip
    '断魂影'天赋需要收集5颗'断魂影之石碎片'合成后使用获得，通过商店购买或者通过特殊目标击杀掉落获得。**暂未实装**

#### 9. 猎影天赋

!!! tip
    灭法之影独有天赋**'猎影'**，通过击杀目标随机获取**本源能量**，到达一定数量后获得**'灭法技能点'**，可通过**唤醒之碑**学习被动或升级更高级的能力。**暂未实装**

#### 10. 唤醒之碑

!!! tip
    将军掉落获得，消耗**'灵魂钱币'**开启。需要获得**'复仇之魂'**、**'强者之魂'**来解锁不同的灭法技能并学习。**'唤醒之碑'**的技能或被动需要消耗上述所说的**'灭法技能点'**来学习。

### 专长说明

#### 基础被动

!!! tip
    具有7个基础被动，为灭法之影专属专长，全部学习后，到达一定等级可获得**「终极·基础被动」**。

#### 轮回乐园系统

- 灭法之影（主职）自带轮回乐园系统，其他职业想要使用需学习专长
- **乐园币**（交易代币）通过击杀目标获取
- **灵魂钱币**（交易代币）12级后击杀目标获取
- **宝箱**：击杀目标有几率获得宝箱，开启后随机获得宝箱品质的物品，小概率获得高一级品质的物品
- **宝箱内包含**：消耗品、武器、护甲、饰品、材料、宝石、强化石、乐园币、灵魂钱币
- **消耗品**：恢复类、提升属性类、血统职业类（获得额外特殊能力，暂未实装）、爆炸物
- **材料**：升级材料（如灵魂结晶等）、食材（用于'夏之季餐厅'增加额外料理）
- **宝石**：用于镶嵌至装备获得特殊属性，无法镶嵌重复的宝石
- **强化石**：强化装备获得额外能力，强化到达一定等级获得额外属性加成，最高强化20级
- **装备**：装备具有评分，当前品质满评分具有额外的特殊属性，少数装备还具有套装属性加成（装备同一套装达到装备数量后获得额外的套装属性）

## 风水师

### 设计理念

!!! tip
    人法地，地法天，天法道，道法自然。一群参悟自然规律，熟练运用自然界元素的修行者。

### 职业定位

!!! tip
    虽然法系职业被定位为辅助，但是亦可根据选择搭配转变为输出。同样输出职业亦可转变为伤害型辅助。

!!! tip
    模组提供了丰富的强力加成和不同强度的子职版本（聚气、观星和寻龙），为了更好的游戏体验，请玩家根据需求自主选择适合的加成和职业版本。

"日曜之力在她左掌心凝结成温暖光团，犹如正午骄阳；月华之气在右指尖流淌为清冷银辉，宛若子夜冰轮。"
