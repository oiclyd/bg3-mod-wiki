# 纷争炼金术

**出处：**纷争系列模组

**关键物品：**炼金日记（书籍）、炼金坩埚（物品）

#### 核心炼制过程 - 白金之尘

1. 秋水仙聚合物 + 颠茄聚合物 = 黑土
2. 黑土 + 独角兽角聚合物 = 白垩
3. 白垩 + 麒麟丝聚合物 = 黄金
4. 黄金 + 匕根草聚合物 = 白金之尘

!!! warning
**合成技巧：**纷争炼金术炼制时只会消耗合成台右侧的物品，把白金之尘放在坩埚左侧即可无限使用同一份白金之尘。
