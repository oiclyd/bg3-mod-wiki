# 专长28：美德之神 - 详细子技能（七美德光环）

- P1 谦卑：24m内友方单位获得攻击掷骰、伤害掷骰增加 2d6 + max(来源角色等级, 来源魅力调整值) / 4。
- P2 宽容：24m内友方单位获得倒地时回复20%生命值。
- P3 忍耐：24m内友方单位获得受伤减免一半。
- P4 勤勉：24m内友方单位获得攻击优势、豁免优势、且豁免检定掷骰不低于10点。
- P5 慷慨：24m内友方单位获得每回合回复 2d8 * max(角色等级, 魅力调整值) 的生命值。
- P6 节制：24m内友方单位获得重击骰子减少5。
- P7 贞洁：24m内友方单位获得AC增加 10 + 熟练加值。
