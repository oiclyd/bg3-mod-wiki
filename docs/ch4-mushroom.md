# 蕈研人

辅助、燃血/中毒、削弱。施法主属性为感知，主要攻击/防御属性为敏捷。法术豁免难度受感知影响，攻击加值受敏捷影响。起始生命值与升级生命值与德鲁伊一致。熟练项：轻甲 / 简易武器 / 敏捷、感知豁免。起始装备：投毒者的长袍。

### 核心名词解释

| 术语 | 类型 | 效果 |
| --- | --- | --- |
| 菌种 | 资源点 | 白露与湿苔根植于此。用于施法，长休恢复。初始上限为2，随等级提升。 |
| 菌落 | 状态 | 每层提供1d4生命恢复（回合结束时触发）。最多叠加3层（6级后提升），战斗结束清零。 |
| 蜜环菌 | 状态 | 每回合开始时体质豁免失败增加1层，叠加至7层时目标获得"易损"状态，随后清空。 |
| 易损 | 状态 | 获得物理、毒素伤害易伤，持续至受到下一次攻击，期间无法获得"蜜环菌"。 |
| 缠绕 | 状态 | 无法移动，回合开始时获得一层"蜜环菌"。 |
| 幽灵菇 | 状态 | 每回合开始时增加1层，叠加至10层时目标获得"明辨"状态3回合，随后清空。 |
| 明辨 | 增益 | AC+2，攻击掷骰+2，法术豁免难度+2，重击阈值-1，期间无法获得"幽灵菇"。 |
| 麻痹 | 状态 | 目标失能，每回合可进行体质豁免检定，成功则解除状态。 |
| 中毒 | 状态 | 攻击检定和属性检定具有劣势。 |
| 污染 | 状态 | 中毒的强化版：除劣势效果外，回合开始时受到1d8毒素伤害，体质豁免失败会使周围3m内的敌人陷入污染状态2回合。 |
| 基因畸变 | 状态 | 攻击检定和属性检定具有劣势，使用动作和回合开始时受到1d10毒素伤害（无视免疫），体质豁免失败则增加1层蜜环菌并传播给周围6m内的敌人，使其陷入基因畸变状态2回合。 |
| 腐朽核 | 资源点 | 战斗中获得，可作为"菌种"使用。上限为3，战斗结束清零（12级后可保留）。 |
| 剑刃防护 | 增益 | 提供对钝击、穿刺和挥砍伤害的抗性。 |
| 薪血 | 技能 | 消耗1d4生命（无视免疫），给予目标4d4生命恢复，并有50%概率给予1层"幽灵菇"。 |
| 菌丝入骨 | 增益 | 获得8临时生命，AC-2，并给予反应技能"菌网感染"。 |
| 菌网感染 | 技能 | 当受到伤害时，使伤害来源目标获得"麻痹"4回合和1层"蜜环菌"。 |
| 菌丝蔓延 | 增益 | AC+3，且每次造成毒素伤害时，目标必须过体质豁免检定，否则陷入"中毒"状态3回合。 |
| 生命共颤 | 增益 | 当其中一方受到伤害时，另一方替代受到1d6伤害。当一方因"菌落"恢复生命时，另一方恢复1d4生命。 |
| 感染 | 增益 | 造成武器伤害时，额外造成2穿刺伤害，目标体质豁免失败则增加1层"蜜环菌"。 |
| 修复孢子 | 增益 | 当目标生命为零时，恢复其所有动作资源，恢复2d4生命，并给予2层"幽灵菇"。 |
| 母根系 | 技能 | 消耗1附赠动作，消耗8%最大生命，传送到所选敌方单位1.5m范围内。 |
| 汲取过量 | 状态 | 无法被治疗，但具有死亡豁免优势。 |
| 赤菌领域 | 领域 | 身处其中的友方目标每回合额外增加1"幽灵菇"，同时蕈研人获得全属性抗性，但每回合结束时扣除自身15%最大生命。 |

#### 1级 基础能力

- **核心属性**：施法主属性为感知，主要攻击/防御属性为敏捷。法术豁免难度等级受感知影响，攻击加值受敏捷影响。
- **起始生命值与升级生命值**：与德鲁伊一致。
- **熟练项**：护甲熟练项：轻甲 / 武器熟练项：简易武器 / 豁免熟练项：敏捷、感知。
- **起始装备**：投毒者的长袍。
- **解锁资源点**：菌种 - 白露与湿苔根植于此。用于施法，长休恢复。
- **解锁被动**：真菌循环机 - 在战斗中，场上每存在一个蘑菇生长完毕时获得1层菌落，在自身回合结束时，每有1层菌落则获得一次1d4的生命恢复，最多叠加3层菌落。战斗结束后菌落数目清空。
- **解锁法术**：看护于躯壳 - 消耗1动作，使自身受到10%最大生命的黯蚀伤害（无视免疫）。选择自身18m半径范围内一名敌方目标，降低其移动距离3m，且在体质豁免失败时为其施加1层蜜环菌。
- **解锁法术**：绒毯上缱绻 - 消耗1附赠动作，消耗1菌种，消耗专注位，以自身9m半径范围选中的目标区域为中心，释放一个半径为3m圆形范围的菌丝藤蔓，属于劣势地形，敌方目标在上方移动时过一个敏捷豁免检定，失败则被缠绕。专注期间，可以在快捷技能栏中选择白绒毯，只消耗1附赠动作后再次释放。

#### 2级 职业技能

- **解锁职业技能**：勉励以精神 - 仅战斗中使用，消耗1动作，消耗10%最大生命+1d4的生命值。选中自身半径9m圆形范围内的2名友方单位（可以选中自己），给予蕈研人自身12%最大生命的临时生命。并给予选中目标1层幽灵菇。

#### 3级 子职分支

#### 星火不息之种（燃血）

- 资源点菌种提升至3
- **解锁职业技能**：浊血饲白花 - 消耗1附赠动作，可选自身消耗5%/10%/20%最大生命，使自身与9m/12m/18m半径圆形范围内所有盟友获得1d4生命恢复和剑刃防护2回合，且受影响目标有25%/60%/100%概率获得1层幽灵菇。
- **解锁职业技能**：哺众于腐壤 - 仅在战斗中可用，消耗1动作，消耗1菌种，消耗20%最大生命，给予自身9m半径圆形范围内所选的2名盟友快捷栏技能薪血持续至使用。战斗结束时去除技能薪血。

#### 石中绿，林中我（中毒）

- 资源点菌种提升至3
- **解锁法术**：以骨寄菌脉 - 消耗1附赠动作，消耗2菌种，给予自身及18m半径圆形范围内2名选中的友方目标菌丝入骨持续5回合。
- **解锁法术**：朽木孕千蕈 - 消耗1动作，消耗1菌种，选中自身18m半径圆形范围内的单个友方目标，给予菌丝蔓延5回合。

#### 4级 共同能力

- **两子职共同解锁职业技能**：故事的根须 - 消耗1附赠动作，消耗1菌种，选择自身18m半径圆形范围内1名友方目标，与其建立链接生命共颤3回合。

#### 5级 进阶技能

#### 星火不息之种（燃血）

- **分支1**：菌物攘赤露 - 消耗1动作，消耗5%最大生命，扣除所有菌落，为自身9m半径圆形范围内所有友方单位提供菌落数目 x 蕈研人自身15%最大生命的生命恢复。
- **分支2**：白斑淬朽刃 - 消耗1动作，消耗5%最大生命，消耗1菌种，选择3m半径圆形范围内3名友方单位，为其武器提供感染10回合。

#### 石中绿，林中我（中毒）

- **分支1**：菌肉的罅隙 - 消耗1动作，选中自身18m半径圆形范围内的尸体（目标尸体不能损毁太严重），目标尸体损毁，尸体所在地表产生6m半径圆形的菌丝藤蔓。
- **分支2**：菌肉的嘤咛 - 消耗1动作，释放3枚毒素孢子，每发毒素孢子使自身9m半径圆形范围所选单位过一个体质豁免检定，失败则受到1d10毒素伤害。

#### 6级 核心被动

#### 分支1

- **解锁被动**：蕈网织静默 - 进入战斗时获得1菌落。

#### 分支2

- **解锁被动**：淤泥更护花 - 在战斗中，每有生物死亡，则蕈研人自身获得1枚腐朽核，上限为3。在战斗结束时腐朽核清零。

#### 7级 资源提升

两子职资源点菌种均提升至4。

#### 分支1

- **解锁职业技能**：白绒拥春雨 - 消耗1动作，消耗3菌种，给予自身9m半径圆形范围内所选单个友方单位修复孢子3回合。

#### 分支2

- **解锁职业技能**：瘴疠覆草野 - 消耗1动作，消耗3菌种，选中区域释放一团3m半径圆形范围的孢子云2回合，孢子云中的敌人攻击掷骰劣势，孢子云中的友方单位额外获得1动作资源，孢子云对蕈研人自身不生效。

#### 8级 子职专精

#### 星火不息之种（燃血）

- **解锁被动**：血液亦温床 - 每当扣除血量的时候，给予自身9m半径圆形范围内的所有盟友3临时生命，可叠加。

#### 石中绿，林中我（中毒）

- **解锁被动**：芽孢入骨髓 - 你所造成的中毒效果增强，更名为污染。

#### 9级 子职强化

#### 星火不息之种（燃血）

- **解锁被动**：命网系群生 - 在战斗中自身回合第一次开始时给予32m半径圆形范围内所有友方单位快捷栏技能母根系持续至使用。战斗结束时去除技能母根系。

#### 石中绿，林中我（中毒）

- **解锁被动**：腐殖化兵装 - 你的所有攻击额外增加2d4毒素伤害。

#### 10级 职业巅峰

#### 分支1

- **解锁职业技能**：结痂的孢尘 - 消耗1动作，消耗10%最大生命，消耗1菌种，消耗1菌落。使自身9m半径圆形范围内两名选中盟友均获得1菌落。

#### 分支2

- **解锁职业被动**：育种致丰饶 - 资源点菌种上限提升至8。

#### 11级 终极能力

#### 星火不息之种（燃血）

- **解锁职业技能**：猩臭化菌杆 - 消耗4菌种，消耗1动作和1附赠动作。创造一片18m半径圆形范围的赤菌领域5回合，身处赤菌领域的友方目标每回合额外增加1幽灵菇，同时使自身获得全属性抗性，但每回合结束时扣除自身15%最大生命。当赤菌领域持续回合结束时蕈研人恢复50%最大生命，但当蕈研人失能时赤菌领域则提前中断，且自身获得汲取过量2回合。

#### 石中绿，林中我（中毒）

- **解锁职业技能**：污染化孕育 - 消耗1动作和1附赠动作，消耗4菌种，引爆自身32m半径圆形范围内所有单位的中毒和污染状态（若同时存在则将二者剩余回合数目相加）变为基因畸变。

#### 12级 终极被动

#### 分支1

- **解锁被动**：深植根循环 - 菌落上限增加至8，每层恢复量提升至1d8。

#### 分支2

- **解锁被动**：腐朽核糜烂 - 腐朽核在战斗结束后不再清零。
