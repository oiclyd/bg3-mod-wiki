# 死亡之歌

!!! warning "核心道具"
    死亡之歌套装升级都需要用到"负能量棱镜"，获取地点：幽暗地域 奥法高塔地下室。

!!! info "升级材料"
    灵魂钱币、地狱铁、金锭、秘银矿石、嗜血灵药、死去的妖精以及死亡硬币。死亡硬币由特定的boss生物掉落。建议打上更多灵魂钱币类似的mod。

!!! success "黑卫专属"
    如果是黑卫职业开卡，并且打了相关补丁，开局即可获得三件死亡之歌装备：凯瑟里克的盔甲、米尔寇信徒手套、米尔寇信徒靴子。

#### 合成武器（需负能量棱镜）

| 武器 | 合成配方 |
| --- | --- |
| 多骨杖（长棍） | 鬼婆的法杖 + 灵魂钱币 + 灵魂钱币 |
| 严峻论点（长柄刀） | 至上真神之视 + 秘银矿石 + 轻语树皮 |
