# 扩容设备

#### 维修套件

- **旧卓尔皮靴（精良）：**装备修补包 + 优质的蛛丝 + 旧卓尔皮靴
- **卓尔皮手套（稀有）：**装备修补包 + 优质的蛛丝 + 卓尔皮手套（精良）
- **蛛丝之靴（稀有）：**装备修补包 + 优质的蛛丝 + 蛛丝之靴（精良）
- **分崩离析的伏形夜影（传奇）：**装备修补包 + 优质的蛛丝 + 分崩离析的伏形夜影（珍奇）

#### 物品组合

- **月光之刃（珍奇）：**破碎魔法剑 + 月光之刃（稀有）
- **精致的雕花铃铛（传奇）：**濒死的妖精 + 精致的雕花铃铛（珍奇）
- **宗主的庇护匕首（传奇）：**吵闹的灵魂钱币 + 宗主的庇护匕首（珍奇）
- **歪曲的渴望（稀有）：**鬼婆的头发 + 歪曲的渴望（精良）
