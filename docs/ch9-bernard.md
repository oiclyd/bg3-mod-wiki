# 进化的伯纳德（BernardEvolved）

按照任务流程走完，可以召唤伯纳德，解除召唤他会在营地中出现，可以交流对话求抱抱。

### 宝宝教程

合成需要四个物品：橙色核心、橙色齿轮、伯纳德的造物之光和橙色的构装体弩。

#### 1. 橙色核心

**获取地点：**奥法高塔地下室中，会看到一个未完成安装的机器人构件。

**具体位置：**以这个构件为原点，他的右侧是最底层电梯，电梯再靠右有一个木桶，桶子上有一个未完成的核心（绿色品质），按住ALT也会显示。

**升级方法：**拿到这个核心，前往高塔顶层，杀掉四个小机器人，核心会自动吸收机器人的能量，具体表现为死掉的机器人会有一束蓝光飘到角色身上。吸收四个机器人的能量后，核心会变为橙色。

#### 2. 橙色齿轮

**获取地点：**未完成的齿轮构件（绿色品质）在顶层的三排石头架子上。

**升级方法：**拿起来后，前往高塔一层，把他和轻语花一样放在装置中，点击合成，得到橙色齿轮。

（这个装置就是你往里放轻语花合成后，可以导致高塔所有奥术炮塔失效的那个装置）

#### 3. 造物之光

这个是个人应该都能找到，毕竟是原版的东西，就是伯纳德的武器。

#### 4. 构装体弩

**获取地点：**在奥法高塔一层门外，轻语树的左侧的一片浅滩，很明显，按ALT就能看到高亮的橙色弩。

### 合成步骤

最后前往地下室，将四个物品合成。记住一定要先拿友好对话戒指，再去顶层对话！不然就激活不了了。

同样，如果伯纳德和你在一起时获得 **波尔** 的祝福，他也会获得。

### 永久升级

**第一章幽暗地狱-精金熔炉：**在装有武僧护符的宝箱的小岛上，可以找到一块坚钢锭，伯纳德可以捡起它。如果伯纳德持着它时被过热，会解锁永久升级。

**第三章：**让伯纳德与一块钢铁守护者掉落的强化炼狱合金互动，伯纳德会捡起它。如果伯纳德持着它时被过热，会解锁永久升级。
