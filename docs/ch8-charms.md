#### 12符咒任务线详细流程

!!! info
**任务主线：**12符咒不仅是散落收集品，还串联了“异界大法师的现世传承”“神器的苏醒”“八大恶魔”“神明信物”“魔气回收计划”“驾驭山之魔气”和“散落的符咒”等连续任务。飞书流程图中的评论已补充到下列步骤中。

#### 异界大法师的现世传承

1. **找到上卷：**上卷在德鲁伊林地医师桌子上。
2. **寻找下卷：**下卷在染病村落死灵书地窖。
3. **收集材料：**第一章林地商人、幽暗地域大地精和白骨斗篷、婶婶、养鸡人军需官均有售卖相关材料，附近也能找到零散材料。
4. **关键材料提示：**蝾螈干在幽暗地域大地精桌子上；河豚干在枭熊洞；魔法香料需要使用染病村落地窖的魔法坩埚合成。
5. **制作法器：**使用河豚干或蝾螈干任意一种，右键合成放入短棒、魔法香料，完成后关闭任务。

#### 神器的苏醒与八大恶魔

1. **阅读书籍：**相关书籍在染病村落死灵书地窖。
2. **寻找宝盒：**潘库宝盒在染病村落死灵书旁边。若没有阅读书籍就拾取宝盒，需要先进行调查。
3. **激活宝盒：**右键潘库宝盒，放入魔法香料进行激活。
4. **寻找恶魔之门：**恶魔之门在晋升之路。靠近时会弹出提示：“你背包里的潘库宝盒正发出震耳欲聋的嗡嗡声，表面烫得惊人。前方的空间正不自然地扭曲着，空气中弥漫着令人窒息的硫磺与泥土气息。”
5. **进入八大恶魔流程：**持有未激活宝盒且未持有魔气收集器时会关闭任务；持有激活宝盒并击败波刚后，若持有魔气收集器则关闭对应任务。

#### 神明信物与波刚

1. **阅读书籍并寻找材料：**书籍在染病村落死灵书地窖。
2. **材料来源：**战鼓可在德鲁伊林地吟游诗人处、地精营地柱子附近获得；另需魔法香料。
3. **制作神明信物：**制作完成后前往击败波刚。神明信物可防止波刚每回合再生血量和护盾，并免疫吞噬。
4. **任务结果：**击败波刚后任务完成并陷入沉寂。

#### 隐藏：魔气回收计划

1. **接取任务：**阅读书籍后接取任务，书籍在染病村落地下室封魔瓶左手边桌底。
2. **制作魔气收集器：**收集封魔瓶、地狱铁、魔法香料后合成。
3. **吸取魔气：**寻找目标 Boss，等待其虚弱时启动魔气收集器。具体做法是让目标血量低于 5% 时，使用魔气收集器击杀。
4. **成功结果：**成功吸取魔气后关闭任务，并解锁“驾驭山之魔气”。若没有吸取魔气，则直接关闭任务。

#### 驾驭山之魔气

1. **阅读哨站书籍：**接取任务后前往幽暗地域塞伦涅哨站阅读书籍。
2. **寻找封魔铁站：**前往精金熔炉岔路口，也就是前往熔炉守卫路上的分叉口，找到封魔铁站及相关书籍。封魔铁站与书籍在一起。
3. **收集制作材料：**阵图会在阅读书籍后掉落；重甲模具与书籍放在一起；精金矿在秘银矿附近；还需要魔法香料。
4. **制作波刚甲：**收集材料制作波刚甲。注意：需要山之魔气才能装备。
5. **完成：**制作完成后关闭任务。

#### 散落的符咒

1. **触发：**调查石符前，如果未阅读对应书籍，会进入“未阅读 / 调查石符”的提示流程。
2. **阅读后收集：**阅读书籍后寻找并收集散落的符咒。书籍在染病村落死灵书地窖。
3. **收集目标：**依次收集十二生肖符咒，流程图中标注为从鼠、牛一直到猪。
4. **归位：**完成收集后进行“十二生肖归位”。
