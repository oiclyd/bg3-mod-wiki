# 种族攻略（部分）

## 种族攻略（部分）


### 谪仙种族详解


**修仙宝术 - 时光洪流**：洪荒时代，大道煌煌，众生皆可悟道修行；末法时代灵气枯竭，唯天命者可修。


### 分神术（5种）


- **夺灵**：夺取觉醒灵根且无法反抗的类人生物的灵根
- **担山**：操控敌人随意摆弄，扔地上倒地1回合，沉重物体造成更大伤害
- **掌日**：消耗神识上限，操控傀儡（神识由感知加值决定）
- **内观**：查看自身资质、大道、灵根、道行
- **洞观**：查看范围内非友方灵根和道行等状态


### 道行系统


- **1-4级** 初入仙途，引气入体（练气前期到练气巅峰）
- **5-8级** 筑基入道，脱胎换骨
- **9-12级** 金丹入腹，威震一方，腾云驾雾
- **13-20级** 破丹成婴，生杀予夺，元神出窍
- **21-40级** 碎元婴而元神显，掌一方天地
- **41-60级** 炼虚期，虚怀若谷炼造化
- **61-80级** 合体期，万法归元
- **81-95级** 大乘期，天人交感证混元
- **96-99级** 九渡雷劫淬仙体
- **100级** 勘破红尘，褪凡成仙


道行在长休和短休后增长，低于一年的道行不提供增益。不同大道有不同快速增长方式。

### 真火炼器


**炼器三要素：**


- **第一 炼器之根-火焰**：谪仙最开始只拥有凡火（1D4），某些天赋异禀解锁天火灵根可以让此火焰效果翻倍。第一章螺壳舰中boss札尔克携带着1D6的幽冥鬼火。
- **第二 炼器之体-器胚**：选择一件武器或者装备作为炼器的本体。必须是唯一性物品。装备分词条装备和技能装备两类，词条装备为最佳器胚。
- **第三 炼器之魂-养分**：每一件带有自己能力的武器装备都可以成为炼器的养料。词条可以完美炼化提取后附着胚体，技能可以炼化提取强化提升技能类胚体。


游戏有数种神火，分别位于螺壳舰扎尔克、拉斐尔、复仇熔炉复仇侍卫、凯瑟里克索姆、安苏身上。击败后用真火灼烧获取。


**炼器步骤**：真火灼烧材料 -> 淬火达到程度失去材料获得"炼器就绪"（持续=境界值+1回合）-> 真火灼烧胚子到过热 -> 再灼烧一次完成。胚子需为唯一性装备。


**淬火需求**：普通12/精48/稀有108/珍奇192/传奇300层，宝材365层。


**炼器高级规则：**


- 获得炼器就绪状态后需在规定回合内将被动转移到胚子上，否则炼器失败。获取被动时，会根据已炼化的装备弹出抉择选项，你可以决定是否保留已炼化的被动。可保留的词条数等同于此时你剩余的"炼器就绪"层数，向下取整。
- 目前只可以将mod装备作为材料而不可以作为胚子，作为胚子无效。且作为胚子的装备需为唯一性装备。
- 火灵根使用真火时，额外施加淬火效率。

#### 炼丹采药技巧


**采药小技巧：**非致命开关下击倒敌人可以摸一次尸体，再击杀之后可以再摸一次尸体，可以让爆率翻倍。

#### 炼妖真解


**炼妖真解：**天地异种，皆蕴造化。以真火淬其尸骸，可炼本命精华。


- **眼魔**：5级眼魔可以炼化金色眼眸，9级黑色眼眸
- **牛头人**：9级牛头人可以炼化大力铁牛角
- **邪魔、提夫林、牛**有概率掉落妖生角，也可进行炼化


炼妖有一定等级限制，小妖无精，大妖华华，具体等级请自行探索。


**炼妖淬火层数需求：**7 x 7 x 等级


#### 宝材器纹详解


**大力铁牛角（器纹-铁角）：**


- 每当你获得临时生命值时，徒手攻击和武器伤害获得等同于临时生命值的加值
- 每当你获得临时生命值时，护甲等级额外获得+2
- 当你使用动作如潮时可以额外获得1个动作


**妖生角（器纹-妖角）：**


- 每回合一次，当你施展一个法术时，徒手攻击和近战武器的伤害掷骰+1并获得1点临时生命值，效果随法术等级提升
- 每回合一次，当你进入荒野形态时，伤害+1受到的伤害-1
- 法术位上限额外提升3点
- 攀上新高：法术位上限额外提升3点


**黑色眼眸（器纹-黑眸）：**


- 被这把武器击杀的生物会在一回合后转化为不死生物为你战斗
- 战斗中每回合结束时最大生命值提升1点，效果随等级提升
- 你控制的不死军团的攻击和最大生命值获得提升，效果随等级提升


**金色眼眸（器纹-金眸）：**


- 圣盾：下一次受到的伤害减少至1
- 发动武器攻击后，获得圣盾
- 获得圣盾时，若已有圣盾，则近战武器的伤害掷骰+1
- 当你治疗一个生物后，使其获得圣盾


**龙珠（器纹-龙珠）：**


- 武器攻击额外造成伤害
- 过量治疗提升器纹·龙珠的额外光耀伤害的骰面，最多提升4次
- 过量治疗：造成光耀伤害时，最小伤害骰+1，最多提升3次


**铁中血（器纹-铁血）：**


- 死亡轮盘：锁定一个目标，使其五回合以后即死，期间自身无法攻击和移动
- 发动武器攻击时损失自身10点生命值并提升等量伤害
- 当你在你的回合被攻击后，回溯该伤害，并获得+1伤害
- 施展一道法术时，你可以以消耗生命值为代价代替法术位消耗，每一环法术位消耗增加10%最大生命值


**玉垂牙（器纹-玉牙）：**


- 每回合一次，武器攻击造成重击时，恢复偷袭骰
- 每回合一次，武器攻击造成重击时，恢复移动速度
- 每回合一次，武器攻击造成重击时，恢复附赠动作


**震雷骨（器纹-雷骨）：**


- 法术迸发：下一次武器攻击伤害触发三次（武器部位）
- 法术迸发：下一次攻击额外造成伤害（护甲部位）
- 法术迸发：下一个法术额外获得两个可瞄准目标（饰品部位）


#### 合欢道修仙小贴士


**合欢道修仙小贴士：**征服可以消耗道行直接征服一个道行比你低的敌人为你所用成为你的仆从，且可以随你传送。用的好妙用无穷，切记勿要征服关键剧情人物，极易翻车！


### 死亡遗产详解（起源-谜团&种族-死生者）


起源谜团与种族死生者在血量低的时候可能会疯狂，这个被动无法去掉，介意就别选这两个。


谜团从教程宝箱拿专属戒指可以用队友临时移除疯狂，也可以点封神的专长免疫疯狂。


邪念起源+死生者种族 = 以提夫林为基础外观的强化版邪念，带额外初始能力和剧情能力。


谜团起源+任意种族 = 邪念起源+死生者种族。谜团可以关闭邪念主线（开就是邪念，关就是塔夫），但别频繁开关。

### 封神之路


共28个专长，通过杀敌获取灵魂 -> 封神之基 -> 神力等级的进阶体系。


### 专有名词


- **一人份灵魂**：10+2*max(力量调整值, 魅力调整值)层
- **灵魂献祭**：战斗外对1HP活物释放，获取30人份灵魂，每长休一次
- **封神之基**：每5000灵魂=1层，最多16层。每层：全属性+2、攻击+1、AC+1、先攻+1、移动+3m、跳跃+1.5m、生命+20

#### 12符咒任务线详细流程


> **任务主线：**12符咒不仅是散落收集品，还串联了“异界大法师的现世传承”“神器的苏醒”“八大恶魔”“神明信物”“魔气回收计划”“驾驭山之魔气”和“散落的符咒”等连续任务。飞书流程图中的评论已补充到下列步骤中。


#### 异界大法师的现世传承


1. **找到上卷：**上卷在德鲁伊林地医师桌子上。
2. **寻找下卷：**下卷在染病村落死灵书地窖。
3. **收集材料：**第一章林地商人、幽暗地域大地精和白骨斗篷、婶婶、养鸡人军需官均有售卖相关材料，附近也能找到零散材料。
4. **关键材料提示：**蝾螈干在幽暗地域大地精桌子上；河豚干在枭熊洞；魔法香料需要使用染病村落地窖的魔法坩埚合成。
5. **制作法器：**使用河豚干或蝾螈干任意一种，右键合成放入短棒、魔法香料，完成后关闭任务。


#### 神器的苏醒与八大恶魔


1. **阅读书籍：**相关书籍在染病村落死灵书地窖。
2. **寻找宝盒：**潘库宝盒在染病村落死灵书旁边。若没有阅读书籍就拾取宝盒，需要先进行调查。
3. **激活宝盒：**右键潘库宝盒，放入魔法香料进行激活。
4. **寻找恶魔之门：**恶魔之门在晋升之路。靠近时会弹出提示：“你背包里的潘库宝盒正发出震耳欲聋的嗡嗡声，表面烫得惊人。前方的空间正不自然地扭曲着，空气中弥漫着令人窒息的硫磺与泥土气息。”
5. **进入八大恶魔流程：**持有未激活宝盒且未持有魔气收集器时会关闭任务；持有激活宝盒并击败波刚后，若持有魔气收集器则关闭对应任务。


#### 神明信物与波刚


1. **阅读书籍并寻找材料：**书籍在染病村落死灵书地窖。
2. **材料来源：**战鼓可在德鲁伊林地吟游诗人处、地精营地柱子附近获得；另需魔法香料。
3. **制作神明信物：**制作完成后前往击败波刚。神明信物可防止波刚每回合再生血量和护盾，并免疫吞噬。
4. **任务结果：**击败波刚后任务完成并陷入沉寂。


#### 隐藏：魔气回收计划


1. **接取任务：**阅读书籍后接取任务，书籍在染病村落地下室封魔瓶左手边桌底。
2. **制作魔气收集器：**收集封魔瓶、地狱铁、魔法香料后合成。
3. **吸取魔气：**寻找目标 Boss，等待其虚弱时启动魔气收集器。具体做法是让目标血量低于 5% 时，使用魔气收集器击杀。
4. **成功结果：**成功吸取魔气后关闭任务，并解锁“驾驭山之魔气”。若没有吸取魔气，则直接关闭任务。


#### 驾驭山之魔气


1. **阅读哨站书籍：**接取任务后前往幽暗地域塞伦涅哨站阅读书籍。
2. **寻找封魔铁站：**前往精金熔炉岔路口，也就是前往熔炉守卫路上的分叉口，找到封魔铁站及相关书籍。封魔铁站与书籍在一起。
3. **收集制作材料：**阵图会在阅读书籍后掉落；重甲模具与书籍放在一起；精金矿在秘银矿附近；还需要魔法香料。
4. **制作波刚甲：**收集材料制作波刚甲。注意：需要山之魔气才能装备。
5. **完成：**制作完成后关闭任务。


#### 散落的符咒


1. **触发：**调查石符前，如果未阅读对应书籍，会进入“未阅读 / 调查石符”的提示流程。
2. **阅读后收集：**阅读书籍后寻找并收集散落的符咒。书籍在染病村落死灵书地窖。
3. **收集目标：**依次收集十二生肖符咒，流程图中标注为从鼠、牛一直到猪。
4. **归位：**完成收集后进行“十二生肖归位”。

### 生离死别


#### 月歌套装说明


月歌套装所有装备升级都需要一个名为“剑舞者的热情”（剑舞热情）的资源，此资源需要通过武器“生离死别”获取。


全套装备包括武器、头盔、斗篷、盔甲、手套、靴子，还有一本讲解故事的书。此套装不算强制绑定，没有套装奖励，但一起使用可能效果更好。


#### 剑舞热情获取机制


- 装备“生离死别”后会获得两个动作和一个被动，其中“剑舞者的优雅”既有被动也有同名动作。
- 开启后进入战斗会自动消耗附赠动作施放“剑舞者的优雅”。
- 战斗中的层数决定战斗结束后转化为“剑舞热情”的数量。
- 开始此动作时获得 3 层；每次成功的近战攻击获得 2 层；命中的法术获得 1 层；闪避对你进行的攻击和法术获得 1 层。
- 被命中时失去 1 层，你的攻击未命中也会失去层数。如果战斗中层数掉为 0，则此次战斗不会获得“剑舞热情”。

### 维克那法袍《魔典》


#### 起始与魔典


开始旅程需要找到《魔典》，位置在染疫村落地下放着《塞尔死灵法术典籍》的桌子上。


书中暗示了长袍第一个位置，在第二章的某个地方。


尝试阅读《魔典》会进行感知豁免。失败会获得一个法术“萃取神圣不死精华”，对游戏中的凯瑟里克的使徒形态使用，提取一种材料来升级长袍。但是如果成功了则会被诅咒，可以在长休后尝试重新阅读。


#### 二阶段


穿着长袍并且背包里拥有“神圣不死精华”时，施放“解封可怖礼服”法术。


第一次升级后，再阅读一次《魔典》且豁免失败，会获得下一个法术“饮下强效诅咒”。


“饮下强效诅咒”只能对一个目标释放：卡扎多尔宫殿里携带吸血咒的维多利亚尸体。一旦施放法术，诅咒就会从尸体中移除，并由你自己吸收。这个强力版本持续 6 回合，每回合会对你造成最大生命值 1/4 的不可阻挡伤害。如果你能幸存下来且不倒地，将获得最终升级法术；如果没能活下来，长袍就永远升级不了。


#### 三阶段


穿着维克那长袍时施放“升阶可怖礼服”法术。该法术会在你承受强效诅咒 6 回合且未倒地后获得。


#### 联动


- **死亡之歌：**长袍算作死亡之歌套装的一部分，并获得套装加成。
- **恐惧主宰：**作为死亡之歌护甲，它允许你在重甲上使用灵魂之盾。
- **维克那法眼：**如果装备了维克那法眼，你的低语者敏捷资源会在短休时刷新。
- **维克那魔掌：**如果装备了维克那魔掌，低语者敏捷允许你每回合以免费动作释放一次升级法术或戏法。
- **亡灵之路：**如果你获得了巫妖身份，维克那的升华法袍现在还会赋予你“法术防范”被动。

同样，如果小趴铲和你在一起时获得 **波尔** 的祝福，她也会获得。


#### 隐藏奖励与能力


现在你可以通过对话将有趣的物品交给小趴铲。如果你拥有能为其解锁某项能力的物品，对话中会出现一个选项，打开一个捐赠窗口（类似给盖尔魔法物品时弹出的窗口），其中会筛选出那些有趣物品。捐赠物品时小趴铲会既兴奋又感激；但如果你什么也不捐就关闭窗口，她会说你是个骗子。


1. 如果 **阿斯代伦** 咬了 小趴铲，她的 **深渊蝙蝠** 将获得黯蚀伤害加成，并在造成黯蚀伤害时恢复生命值。同时解锁「骗子」背景目标。
2. 如果 小趴铲 消耗掉她棺材里找到的 **铲粪器**，她就能学会 **粪便云**（相当于匕首之云）。你也可以把这把匕首扔向她，她同样能学会该法术。如果匕首命中并伤害到她，后一种方式也可能解锁「诡术课」背景目标。
3. 如果 小趴铲 在第二章的 **治疗之家** 造访 **强酸坑**，她的 **洞穴青蛙** 将学会 **恶臭毒雾**，并在受到攻击时反弹 1d4 强酸伤害。必须走到坑底才能触发。
4. 如果你用涂了 **西索博得的特制穿肠散** 的武器攻击 小趴铲，她的 **相位蜘蛛幼崽** 将学会用毒素攻击施加同类型的毒。解锁「骗子」背景目标。
5. 如果你在 小趴铲 在场时获得 **波尔** 的祝福，她也会同样获得。
6. 如果你将 “碎蛋者”、“朋友”头骨 和 **炼金之火** 组合，你会得到一个物品，为小趴铲解锁一个动作，让她每长休投掷一次手雷。起初它只会造成一些钝击伤害并附带炼金之火效果，但通过观察 **戴摩** 给你的某种手雷可以升级一次，再通过观察 **巴克斯** 给你的某种手雷可以再次升级。
7. 如果你击败了 **蛛化精灵**，小趴铲 可以在其附近使用一个行动“吸收”其知识，让相位蜘蛛幼崽施放非传奇版本的 **蛛网结界**。
8. 如果你在任何章节击败了 **奇怪的牛**，小趴铲 可以在其附近使用一个行动“吸收”其知识。一旦队伍抵达 **上城的高厅**，这将解锁她变身的强化版本。每个新形态都共享原形态的能力，并额外增加一些独特能力。强化形态包括：
          
- **披风怪（强化深渊蝙蝠）** —— 大部分能力继承自深渊蝙蝠，额外增加 幻象（每战斗 3 次，如同第二章）、多重攻击（第二章的削弱版）和 死亡之环（每战斗一次）。
- **相位蜘蛛（成年模型）** —— 能力相同，但伤害提高，蛛网范围更广。同时获得 死云术（每战斗一次）。
- **地龙（强化洞穴青蛙）** —— 实为双脊龙，可以使用龙裔的所有 喷吐武器 以及 哈达之饥渴（每战斗一次）。保留洞穴青蛙的所有被动能力。
- **暗影豹（强化暗影猫）** —— "藏身暗影"不再需要处于遮蔽状态才能生效；猛扑不再需要激怒；偷袭伤害现在与普通游荡者一致（从 +3d6 提升到 +6d6）。撕咬和颈动脉撕割的伤害与黑豹相同。这形态威力十足。


*显然，这些强化形态比普通形态强得多。你并不会失去原始形态，所以如果觉得新形态太强，你完全可以继续使用原来的形态。或者干脆不吸收奇怪牛的知识。也可以为了背景目标吸收知识，但获得法术后直接忽略不用。*
9. 如果你击败了 **蜘蛛女王**，可以吸收她的知识，在相位蜘蛛幼崽形态下解锁“巨型形态”，让 小趴铲 每长休变大一次。在巨型形态下，她将能使用 **爆裂虫群**。
10. 如果小趴铲捡起她棺材里的 “小趴铲的朋友探测器”铲子，她就能像玩家一样挖掘土堆。

如果你在安装这个模组前已经加载了第一幕，铲子棺材里的一些物品可能无法合并。为了帮助你，你可以在营地连续使用疾走三次，解锁一组配置法术，帮助解锁“获取碎蛋器解锁器”，找到卷轴召唤卷轴（如果丢失了），解除战术家限制，并尝试寻找刮屎器（如果你不在第一幕区域或匕首被毁，可能无法使用）。这些法术在你离开时会被移除但你需要时可以重新解锁。

## 难度攻略


### Dota难度


#### 名词解释


- **抗性：**常规怪能拥有的50%减伤的抗性，能用易伤状态抵消
- **免疫：**免疫特定状态或伤害，减伤100%，需要特定buff或专长破除
- **减伤：**高难特有特性，直接减少绝大多数伤害数值
- **真伤：**无视抗性、免疫和减伤的特殊伤害


#### 高难思路


> **注意：**高难构筑与原版完全不同，dota、谪仙、封神三个难度都有减伤。


- 武僧等多段附伤职业几乎沦为地板砖
- 需要保证伤害构成纯粹，不超过三个伤害类型
- 后期敌人人均全免疫，造易伤手段失去意义
- B类附伤和C类附伤效果远高于A类
- 需要堆高先攻一回合清场


#### Dota难度开启建议


> **警告：**前期不要勾选boss，难度不是一个数量级！


- 建议在二级点出购物商店的专长
- 在林地戴摩手上买到自带法术反制和法术无效化结界的法杖之前不要开启

| 属性 | 加成效果 |
| --- | --- |
| 力量+1 | 生命+5，近战攻击+1 |
| 敏捷+1 | 远程攻击+1，攻速+1 |

**攻速说明：**每有一点攻速，有1%概率增加一个标动，一回合只能生效一次，多段攻击会判定多次。

> **通用建议：**所有队友攒150金币买可靠铁铲（最稳定的金币来源）。每一章限定增加两件成品装备（房规推荐）。点金手能更快赚钱，但来钱太快不太建议出。

- **第二件** 永恒之盘

### 动态精英


**动态等级（Scaling）**


#### 1. 自适应等级系统（Adaptive Leveling）


当敌人等级低于玩家等级时，敌人会自动匹配玩家等级；首领（Boss）的等级则额外提升+2。


#### 2. 队伍规模生命值加成（Party Size HP Caps at 8 party members）


- 单人玩家：无变化
- 双人队伍：生命值 +3%
- 三人队伍：生命值 +6%
- 四人队伍：生命值 +15%
- 五人队伍：生命值 +30%
- 六人队伍：生命值 +45%
- 七人队伍：生命值 +60%
- 八人及以上队伍：生命值 +75%


#### 3. 等级缩放生命值（Level Scaling HP）


- 1-3级：生命值增加量为「等级 + 体质修正值」
- 4-6级：生命值增加量为「等级 × 2 + 体质修正值」
- 7-9级：生命值增加量为「等级 × 3 + 体质修正值 × 2」
- 10-12级：生命值增加量为「等级 × 4 + 体质修正值 × 2」
- 13-15级：生命值增加量为「等级 × 5 + 体质修正值 × 3」
- 16-18级：生命值增加量为「等级 × 6 + 体质修正值 × 3」
- 19级以上：生命值增加量为「等级 × 7 + 体质修正值 × 4」


#### 核心属性（Core Stats）


*核心属性严格与等级区间挂钩，与队伍规模无关*


- 1-3级：无变化
- 4-6级：攻击投掷/豁免投掷 +1
- 7-9级：护甲等级（AC）+1，攻击投掷/豁免投掷 +1
- 10-12级：护甲等级（AC）+1，攻击投掷/豁免投掷 +2
- 13-15级：护甲等级（AC）+2，攻击投掷/豁免投掷 +2
- 16-18级：护甲等级（AC）+2，攻击投掷/豁免投掷 +3
- 19级以上：护甲等级（AC）+3，攻击投掷/豁免投掷 +3

### 动态遭遇


该模组让世界对你的选择、玩法风格和行动做出积极反应，引入了一系列精心设计的场景战斗，确保你的决定具有分量和潜在的后果。


#### 当前特性如下：


#### 埋藏之惧


挖掘宝箱现在有12%的概率惊扰亡者，在你的战利品旁直接生成一名亡灵敌人。


#### 亡者印记


杀戮必有后果。击杀平民有15%的概率使其化为复仇之魂，并使你获得"亡者印记"状态。这些灵魂会伺机而动，不久后伏击你。警告：若在伏击触发前有多名平民诅咒你，所有生成的灵魂将同时出现并发动攻击。


#### 元素汇聚


释放原始魔力会吸引元素位面的注意。在单回合内造成超过50点特定元素伤害有8%的概率生成对应的元素生物（火元素例外，生成概率为4%，且需要造成75点以上伤害）：


- 闪电伤害 ➔ 气元素
- 火焰伤害 ➔ 火元素
- 强酸伤害 ➔ 土元素
- 寒冰伤害 ➔ 水元素


#### 吉斯洋基人的猎杀


每当你动用夺心魔能力时，有20%的概率被吉斯洋基人狩猎小队感知到灵能波动，他们会传送过来消灭你的队伍。


#### 力量的代价（诅咒神器）


击杀特定敌人有罕见的1.5%概率掉落六件独特诅咒物品之一，它们会永久赋予你一项属性+1（激活时额外+1），但会招致敌对势力无休止的伏击，试图夺取神器。此外，贪婪诅咒意味着搜刮金币时有0.1%的概率偷偷将第七件神器塞入你的口袋，这会大幅提高商人价格，直到你解除诅咒以获得永久的商人折扣。所有神器都会绑定你的角色，直到施展"移除诅咒"法术。


#### 战场清道夫


血腥场面会吸引食腐者。在上一次遭遇结束约10分钟后，会生成狗头人，剥取亡者的财物。尽快收取你的战利品——他们会带走所有遗留物品，仅保留关键剧情物品。若你在此时间段内进行多场战斗，狗头人会按顺序清理最早的战场。狗头人每具尸体进入2回合的"搜刮"状态，区域被清空后便会消失。


#### 暗影之根


击杀德鲁伊林地中潜入的三名暗影德鲁伊会吸引荒野中其他活跃的暗影德鲁伊的注意。这会触发持续威胁：当你的队伍移动和探索时，敌对的暗影德鲁伊会无情地追踪你的踪迹并伏击你。这种追击会在你旅行过程中有机持续，只有当你抵达幽影诅咒之地时才会停止——那是一片连暗影行者都觉得过于黑暗的地方。（如果你安装了 荆棘之环-Circle of the Thorns 模组，暗影德鲁伊将拥有该模组中的一些法术和能力）。


#### 饕餮之息


熊熊篝火与丰盛餐食是旅途中的必要慰藉，但香气会传播甚远，吸引不速之客。完成一次长休后，有8%的概率你盛宴的香气会吸引一只饥饿的食人魔来到营地。一旦队伍达到6级，有25%的概率第二只饥饿的食人魔也会试图加入这场盛宴！


#### 有灵之钢

#### 动态宿敌系统（Dynamic Encounters）


当敌人生命值降至20%以下时，有3%的概率触发此事件。敌人不会战斗至死，而是突然逃离战场，消失在阴影中，伺机对您的队伍进行复仇。此系统严格排除首领、商人以及绝大多数剧情NPC，以保护游戏进程的完整性。


敌人逃脱后，会开始跟踪您，而且等待伏击的时间越长，其致命性就越强。每次您进入新的战斗时，系统会进行判定，观察追踪者是否会伏击您。在敌人逃脱后的前四次遭遇中，有30%的概率以普通威胁等级伏击您。如果敌人耐心等待至第五至第七次遭遇，将升级为精英威胁，每次遭遇有40%的概率发动袭击。如果敌人一直等待至第八次遭遇，其回归将是绝对的，以100%的概率作为可怕的迷你首领伏击您。


为确保这些遭遇保持平衡，伏击的实际难度严格受限于您队伍中最高等级的角色。如果最高等级角色在1至4级之间，系统将强制把伏击难度限制在普通难度，无论敌人等待了多久。对于5至8级的队伍，复仇可升级至精英难度。一旦任何队伍成员达到9级或以上，安全限制将完全解除，允许敌人成为迷你首领。


当复仇最终降临时，系统会动态投掷独特的强化效果来帮助敌人向您复仇。普通复仇投掷1-3个低级强化。精英复仇增加危险，有50%的概率继承两个低级强化，同时投掷1-3个中级强化。迷你首领有机会同时保留普通和精英强化，并额外获得1-3个高级强化，将之前被遗忘的小卒转变为无与伦比的异类。

#### 运作方式


为使这些遭遇保持平衡与沉浸感，"命运羁绊"系统遵循一套严格的背景规则运行：


- **共享冷却：**每次长休期间，此类遭遇只能触发一次。一旦伏击发生，队伍在长休前将不会受到"命运羁绊"事件的影响。
- **仅限当前队伍：**同伴必须位于你的当前队伍中（营地外）才能触发其专属遭遇。
- **营地记忆系统：**若有多个符合条件的同伴与你一同离开营地，游戏将进行掷骰以选择哪位同伴触发"命运羁绊"事件。若你返回营地后再次与同一同伴离开，目标仍锁定在该同伴身上。若你将其换下，系统将重置并重新选择符合条件的同伴。

### 骷髅挑战


**瓦尔克拉娜-骷髅挑战（1d4+1）-Valkrana's Skeletal Challenge Lite**


会在进战时随机刷新骷髅，默认最多随机5个骷髅，也可以自己调刷新数量，概率。


代码为 `!vsch_config a b c d e`


- **a** 为每次战斗刷新几率，从 0%-100%
- **b** 为每次刷新的最少数量
- **c** 为每次刷新的最大数量，0-100
- **d** 会将骷髅生成设置为 Skeleton Crew（骷髅船员）的静态等级池，而不是平常的缩放池。1最低，3最高，false 会被设置成默认（禁用）
- **e** 若设为 "true"，则范型将仅生成范型骷髅单位，例如死灵代言人植入者·维恩


**举例说明：**


`!vsch_config 52 9 10 3 false`


会将每次遭遇的刷新率设置为 52%，每个刷新有 9–10 个骷髅，并且总是从最大的骷髅池中生成，不考虑队伍等级，不会仅生成原型骷髅。

2. 使用法术“不死之路 - 冥想”（获得“禁忌知识”之后自动解锁）


3. 使用鬼婆的药水“踌躇意志”和“失踪宠物”，直到你成功。注意 “失踪宠物”要后喝


4. 进入长休，在睡觉前自杀。等待10回合自动复活（如果在10回合结束前复活，则变身失败）


5. 完成长休


#### 能力：


1. 被认为是不死生物，不会成为大多数治疗法术的目标，且会受到驱散不死生物和控制不死生物的影响


2. 毒素和黯蚀抗性


3. 对抗魅惑获得优势


4. 对疾病免疫


5. 死亡豁免检定具有优势


6. 获得一个职业动作，可以玷污生物的尸体以恢复生命值（消耗附赠动作）


7. 打断法术 瓦尔克拉娜的法术书-《尸体手榴弹》


#### 模组和角色互动：


1. 邪念（起源）：造成的黯蚀伤害+1（每个亡灵阶段+1）


2. 堕落的阿斯莫（种族）：对黯蚀免疫


3. 死亡之歌套装或死灵师10级（装备和职业）：对黯蚀免疫


4. 同时具备上述两者：+1伤害减免


5. 恐惧主宰（子职）：所有献祭尸体均会触发污秽效果（玷污一具尸体+1d4生命值）


6. PF2E（法术）：伤害术能正常治疗。


7. 黑卫（主职）：黑卫的所有治疗都与亡灵之路兼容


可获取的力量：


1. 获得劳薇塔的灾祸，用特定的法术摧毁它，可获得被动：近战攻击增加1d6范围黯蚀伤害（2m范围，可开关，影响自己和友军）


2. 恐惧主宰：第二章连续4回合受到幽影诅咒的影响。增益：在你不精通的掷骰上增加一半的熟练项加值，包括豁免检定，以及免疫幽影诅咒


3. 幽影獒召唤：接近幽影獒并尝试驯服它（用一个动作来驯服）。如果你成功则会成为一个随从，死去后可以重新召唤。


4. 邪恶斗篷：完成第二幕的惩罚恶人任务。效果可以追溯。（仅当被标记为圣武士时可用，包括黑卫）效果：十字军斗篷为黯蚀伤害。


5. 恐惧主宰：激活属于恐惧主宰职业的骨雕，每2个会增强召唤物的力量，4个骨雕时获得永生之袍如果已经获得了它，则会增强其力量，8个骨雕时背包中拥有可以谐调石头为阿扎顿增强伤害。

#### 第二步：成为吸血鬼


1. 成为亡灵（不死生物）


2. 阅读《黎明之影：吸血鬼学说概论》（在养育间，审判官右边宝藏室）


3. 使用法术“吸血鬼之路 - 冥想”


4. 使用鬼婆的药水“腹中蝴蝶”和“狂乱之吻”，在效果持续期间存活。


5. 使用鬼婆的药水“母亲的憎恶”，然后用给予的法术咬一个活着的类人生物。


6. 长休


#### 能力：


1. 吸血鬼：你对生者造成的任何黯蚀伤害都会恢复你1点生命值；+1.5M移动速度；获得“吸血鬼啃咬”（造成1d4+熟练项加值）；黑暗视觉（12M）；获得“吸血鬼飞行”（8M，需要出发和到达点都为遮蔽）


2. 隐秘威胁：被遮蔽时获得隐藏威胁（-1重击骰，+1DC）


3. 阳光弱点：在阳光下攻击掷骰和豁免检定处于劣势


4. 流水弱点：在水中停留的时间越长，受到的强酸伤害就越高


5. 吸血鬼再生：当不在阳光下且没有被光耀伤害命中时，每回合恢复1点生命值（可切换）


6. 勃艮之悦：踏入血液时，吸收血液以恢复1d4生命值（可切换）


7. 迷魂：对不死盟友和中立者施放此法术，使其在大多数情况中被视为随从（如慑心魔灵光、恐惧主宰特性、死亡之歌戒指等）；同时从目标身上窃取1点所有属性；你一次只能拥有一个奴隶；基础范围为3m；在“吸血鬼”阶段，它只能以中立的不死生物和玩家角色为目标，不能以盟友召唤物为目标。（阿斯代伦因此功能被视为不死生物，其他的则需要以某种方式成为不死生物，如亡灵种族或从瓦尔克拉娜的法术"瓦尔克拉娜的死亡灌注"）。在“高等亡灵”阶段，它获得+1.5M射程，可以对敌方不死生物使用。如果它们威能通过感知豁免检定，就会成为你的奴隶，直至他们死亡或你迷惑另一个生物。在“巫妖”阶段，它又获得+1.5M射程。在中立或敌对目标上，会控制该实体。在盟友目标上，如果他们抵抗了该法术，目标可以心甘情愿地向吸血鬼施法者宣誓效忠（避免在未来被迷惑）。


宣誓效忠：两个角色将不可逆地绑定；


领主获得+1M移动；（吸血鬼阶段）


领主+1 DR；（高等亡灵阶段）


领主对非盟友目标造成+1黯蚀伤害（巫妖）；


宣誓者被标记为亡灵召唤，可能允许某些效果以它们为目标（就像普通奴隶一样）；


领主可以同时拥有多个随从；


他们之间没有属性削弱；


宣誓者不算奴隶，所以领主仍然可以绑定一个奴隶


#### 模组和角色互动：


1. 阿斯代伦：啃咬+1d4黯蚀伤害（像戏法一样）


2. 邪念：啃咬会使目标流血，持续3回合


3. 邪念：当有人因你而流血时，他们的脚下会形成一个小血池


4. 恐惧主宰：施放冥界武装时，你和你周围的盟友在对生者进行伤害性攻击和法术时获得1d4治疗


5. 强化饮命者（邪术师的魔能祈唤）：邪术师的饮命者对生命值＜50%的目标造成双倍伤害


6. 强化吸血鬼之吻（死亡之歌）：如果你在被死亡之歌的剑吸血鬼之吻治疗时已经满血，则获得临时生命值（上限为20）


7. 黑卫：强化职业本身


8. 噩梦：每天晚上，25%几率获得噩梦。有10种，每个都会在当天施加独特的Debuff（注意是Debuff，而不是Buff，成为吸血鬼后可以服用箴言菇来永远防止噩梦）

活埋：当有人在附近使用铲子时，你必须成功进行感知豁免检定，否则就会恐惧地逃跑。如果你是使用铲子的人并且失败了，你会受到 2d8 心灵伤害

#### 可获得的力量：


1. 使用给予的法术摧毁悲伤（卡扎多尔掉的法杖）获得：+1法术DC和法术攻击掷骰；当敌人对你的一个法术进行豁免检定失败时，恢复1d4*（熟练加值-1）；枯萎术（不消耗法术位，每次长休一次）


2. 以吸血鬼的身份进入堕影冥界：+4m吸血鬼飞行；黑暗视觉变为高级黑暗视觉；不再害怕阳光


3. 恐惧主宰：效果同上并增强：+8m吸血鬼飞行；黑暗视觉变为魔鬼视觉；不再害怕阳光；永生之袍根据遮蔽情况获得伤害减免


4. 飞升吸血鬼：从阿斯代伦那里窃取力量。

5. 凯兰沃的天平：在卡扎多尔地牢中发现的饰品（乐器），隐藏物品。

#### 第三步：成为高等亡灵（Greater Undeath）


1. 成为吸血鬼（见上一步）


2. 阅读《巴萨扎尔编年史》（月初之塔巴萨扎尔密室里）


3. 使用法术“高等死灵术的秘密 - 冥想”


4. 使用鬼婆药水“爱人的贪婪”和“枯萎梦境”


5. 长休几次，每次进行一次感知DC20的豁免检定，如果失败了则受到3d8心灵伤害，并做噩梦，且此次长休不算在内，如果成功3此，则试炼完成。


#### 能力：


1. 高等亡灵：免疫毒素和黯蚀；寒冷抗性；免疫魅惑、疯狂、恐吓、发笑和控制不死生物；对光耀、火焰和强酸易伤


2. 你的法术+1d8黯蚀伤害


3. 除非被光耀、火焰、强酸或重击命中，否则你无法倒地。当此效果触发并且重新站起来时，你会恢复你的移动速度和附赠动作，但不会获得动作。


4. 噩梦与梦境：解锁10个新的夜间活动，5个是新的噩梦，5个是好梦（带来增益），其中4个是专属于邪念的，并且取决于你是屈服于黑暗还是抵抗它，如果你服用了箴言菇则阻止噩梦，20%几率触发高等亡灵之夜事件，5%触发吸血鬼之夜事件


噩梦与梦境列表：


虚空低语（噩梦）：当被重度遮蔽时，你在隐匿检定中劣势


被遗忘的倒影（噩梦）：一天之内魅力-1


瘟疫（噩梦）：当天无法使用附赠动作


绿洲（好梦）：最多无视5点伤害，直到你攻击，战斗结束后刷新


和平森林（好梦）：将你的熟练项加值添加到魅力检定中


内心图书馆（好梦）：在奥秘、历史和宗教检定中具有优势


羊群中的狼（好梦）：当敌人进入你的领地时，他们必须成功进行感知豁免检定，否则被恐吓


死亡先驱（好梦）：杀死生物时，你会陷入疯狂直到下一回合结束，你造成额外的物理伤害，并在2此攻击掷骰中获得加值


糖果武器（噩梦）：你的攻击掷骰受到1d6的惩罚，当你未命中时，受到2d6的心灵伤害


内心的邪恶（噩梦）：每分钟，你都会进行一个感知豁免检定，防止在一回合内变得疯狂，直到你满足/高兴


#### 模组和角色互动：


1. 堕落的阿斯莫（种族）：没有光耀易伤


2. 死亡之歌盔甲或死灵师10级（装备和职业）：没有火焰和强酸易伤


3. 同时具备上两者：+2伤害减免


4. 死亡之歌头饰（装备）：对抗命令亡灵的优势


5. 恐惧主宰（子职）：冥界武装可以免费施放


6. 邪念（起源）：当你对一个生物重击时，该生物和附近的任何敌怪必须成功进行感知豁免检定，否则被恐吓，直到下回合结束。如果没有抗性或免疫，则会穿刺易伤。


7. 黑卫（主职）：职业能力加强

1. 使用给予的法术摧毁死灵珍爱法杖（秘术师掉的那把）获得：增强死灵法术，生物在对抗你的死灵法术的豁免检定处于劣势；用法术杀死一个生物时，吸收它的能量获得生命净化，生命精华可用于下一个死灵法术不消耗法术位，生命精华可叠加，一次消耗叠加一层，长休后剩余的层数将被吸收为永久最大生命值（7层换1点）

3. 霸主：在战术家或荣誉模式下，与秘术师的战斗中，击中它也触发它的反应，在被施加状态时，通过豁免检定打破，每回合有一次机会，成功后赋予霸主特性：对支配免疫。
