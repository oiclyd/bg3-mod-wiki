# 自选模组

## MCM数据统计

适合联机看队友数据，单机作用不大。

和完全自定义雇佣兵冲突，我包里没有，有自己装的注意一下。

## 功能-星命

为原版职业和部分mod职业增加星座选择，但这会影响佣兵使用！

## 肉鸽模式

**现已汉化**

必须排序在【前置-7-9环法术-AdvancedTTSpells】的下面

与整合包内的难度模组能产生联动效果

- 正常游玩不要装此模组，游玩肉鸽模式务必开新档。
- 此模组开启后会改变游戏模式，会影响存档。
- 带有自定义物品的模组需要排在他下面，这样物品才有机会掉落作为奖励。整合包玩家排在【前置-7-9环法术-AdvancedTTSpells】下面即可。
