# 风水师

### 设计理念

人法地，地法天，天法道，道法自然。一群参悟自然规律，熟练运用自然界元素的修行者。

### 职业定位

虽然法系职业被定位为辅助，但是亦可根据选择搭配转变为输出。同样输出职业亦可转变为伤害型辅助。

模组提供了丰富的强力加成和不同强度的子职版本（聚气、观星和寻龙），为了更好的游戏体验，请玩家根据需求自主选择适合的加成和职业版本。

### 专长

达到对应等级后获得一次选择机会：4、8、12、14、16、18、20、24、28级。达到等级后可选数量：1

### 本命玄枢

勘天星定命，御三炁通玄（1/6/12级择风、水、气本源之力，引星象入命重塑战局）。达到等级后可选数量：1

### 地脉业术

执罗盘布煞，化万象为用（1级择一地脉秘传，13级起融通十二流派，终成风水大宗师）。达到对应等级后获得一次选择机会：1、13、14、15、16、17、18、19、20级。达到等级后可选数量：1（15、17、19级可选2个）

### 聚气子职

聚气、观星和寻龙子职拥有相同的体系，聚气为**弱化版**，伤害、效果、范围、目标数与武器强度均被削弱。

角色类别：武僧 | 施法者属性：全施法者 | 学习法术：是

**操作提示：**提升感知/法术豁免难度，合理利用技能恢复资源
