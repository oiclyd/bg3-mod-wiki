# 武侠子职合集（WuXiaPackOne）

六个具有浓厚武侠风格的子职业，分别对应牧师、法师、野蛮人、邪术师、游荡者和武僧。另含四件传说级乐器装备。

### 一、日月领域 - 牧师子职业

"日曜之力在她左掌心凝结成温暖光团，犹如正午骄阳；月华之气在右指尖流淌为清冷银辉，宛若子夜冰轮。"

- 1级 日月交替：每回合自动切换日之轮转（光耀伤害附加+免疫黯蚀+AC+3+DC+3+减伤3）和月之轮转（黯蚀伤害附加+免疫光耀+攻击优势+不触发借机+重击-1）
- 2级 菩提清心曲：解除疾病/中毒/麻痹/失明，豁免+AC+3
- 6级 不激不随：附赠动作施展+解锁加速术/迷踪步
- 8级 笑傲江湖曲：18m心灵伤害+震慑

### 二、星宿法师 - 法师子职业

"星宿老仙，法力无边！天下至毒，莫不生发于木；诸般造化，终将臣服于王。"

- 2级 星宿毒功：免疫中毒，毒素无视抗性免疫，毒素攻击优势，连珠腐尸毒
- 6级 化功大法+凭虚御风
- 10级 万蛊之王：每次伤害追加毒素+星宿毒云掌
- 14级 星宿毒环：6m剧毒光环

### 三、太玄道途 - 野蛮人子职业

"谁能书阁下，白首太玄经"

- 3级 太玄之道：太玄之重（重击-1骰+10%最大生命钝击）/ 太玄之轻（不被重击+免疫缠绕等）/ 罗汉伏魔替换狂暴
- 6级 太白剑经：5种招式需装备对应武器解锁
- 10级 沧溟斩：9m范围+4d8冰霜

### 四、越女剑仙 - 邪术师子职业

"牧羊少女，以惊世的剑术刺破历史，又因一份纯粹的钟情，归于寂寂山河。"

- 1级 天元剑意（最多3层：攻击+1/伤害+1d4力场/重击-1）+ 天仙剑影
- 6级 天仙剑阵：敌人攻击队友时自动反击
- 10级 九霄仙息：冲向敌人攻击后对附近9个敌人二次攻击

### 五、小李飞刀 - 游荡者子职业

"兵器谱探花，小李神刀，例不虚发。"

- 3级 飞刀暗器（绑定匕首+1附魔+自动返回）+ 捻熟于心（投掷+1d6+8）
- 4级 飞刀专精（每回合3把）+ 夺命飞刀
- 7级 例不虚发：暴击时生命<100直接击杀
- 9级 飞刀幻影：分裂多枚6d6穿刺
- 13级 暴雨梨花多重：可选3目标

### 六、明教宗 - 武僧子职业

"生亦何欢，死亦何苦？为善除恶，唯光明故。"

- 3级 炽热徒手+混元体质+乾坤大挪移（反应减伤+反击）+ 迷踪步传送版
- 6级 太极圆转：受伤获得太极劲（伤害+1d6钝击+AC+1）
- 9级 圣火令秘法+群体疗愈气劲
- 11级 大光明劲：拉向自己+火焰神圣伤害+缓速
