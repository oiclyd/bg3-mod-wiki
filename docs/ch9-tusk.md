# 进化的长牙（TuskEvolved）

**前置条件：**需要动物交谈。

### 招募方法

1. 与翠绿林地的野猪长牙对话。
2. 和内蒂对话开始"营救德鲁伊哈尔辛"任务后，与其对话。
3. 在背包留下一个能召唤其的物品。

**备注：**如果哈尔辛在第二幕可招募时还没招募长牙，哈尔辛就会获得召唤长牙的物品。第三幕同样有效。

### 哈尔辛的承诺

游侠的动物伙伴里的野猪和长牙只会出现一个。用另一个队友召唤游侠动物伙伴里的猪，再用一个队友召唤长牙，将两只召唤物聚集在一起，30m 内没有队友且不在操控中时，长牙会获得一个永久被动：动物活力。

动物活力的被动恢复量根据游侠召唤的动物伙伴等级而定：1-4 级恢复 1d4，6-10 级恢复 1d6，11+ 级恢复 1d8。在获得 11 级被动后就不需要再重复获得，只会生效最高增益。如果待在一起没有获得被动，尝试操控长牙在另一个动物伙伴边上走一下。

同样，如果长牙和你在一起时获得 **波尔** 的祝福，他也会获得。

### 地狱野猪解锁

可以将一枚灵魂硬币丢在地上，操控长牙使用（看起来像是吞下），或者扔一枚灵魂硬币投向它（确保血量足够）。几秒后弹出"地狱野猪解锁"的提示，可以在狂暴状态下变为地狱野猪（每短休一次）。灵魂硬币不需要每次使用狂暴后重新补充，解锁被动后即可永久获得。

**注意：**如果游戏中任何时候哈尔辛被宣布永久死亡，长牙将不再可召唤并离开营地。
