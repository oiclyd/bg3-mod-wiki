# 进化的萨扎（SazzaEvolved）

**招募条件：**满足获得"放了她！"成就所需的条件，满足后才能被招募。将她从林地、明萨拉和月初之塔中解救出来。解救后，她不会走进幽影诅咒之地，而是会在桥边的路标等待，和她交谈就可以招募。

**备注：**如果在安装此模组后已经进入了第三章，去马戏团的商店对话也会有同样的选项。同样，这只有在满足成就所需的条件才会有。

### 萨扎的宠物

**招募地点：**染疫村落到地精营地门口可以招募小利爪。（萨扎等待你的地方）

**招募方法：**

1. 需要永久击败检查点的所有地精和小利爪的父母。如果警报响起有增援到来，这些生物也必须被击败。
2. 目标是让小利爪成为最后的战斗者。像枭熊幼仔一样，如果只剩它在战斗中，它会脱战，并且不会对你进行攻击，也不会为地精而战，任务日志会有说明。
3. 战斗结束后，它会在附近休息，你需要平复它的情绪，或等待下一次长休。
4. 等它平静下来就会饿。放下任何肉类物品（矮人肉也能用），它会走过去吃掉。
5. 喂饱之后你需要说服它去你的营地，但只有当你身上有它族群气味时，才会听你的。
6. 去狼父母被击败的地方，旁边会有狼粪便。右键点击涂抹在脸上之后（如果点击显示无法使用，换一个人试试），和狼崽对话，它会觉得你是它的父亲，告诉它去营地，任务完成。
