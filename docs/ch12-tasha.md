# 塔莎炼金术

**出处模组：**装备-骨铁饰品-EldertideArmament

**关键物品：**塔莎的炼金术大全（书籍）、巫后护身符（护符）

**获取方式：**炼金术大全在教程宝箱和部分商人处购买，巫后护身符建议用MCM简易作弊获得

**合成方式：**通过巫后护身符给予的二环法术召唤合成台，投入对应炼药素材获得药剂
