# 正义之路

!!! warning "职业限制"
    此套装绑定圣武士职业，紫色及之后非圣武士不能装备。蓝色品质即使装备了但不是圣武士用处也不大。

!!! success "材料来源"
    升级材料全部来自纯善线任务奖励，如果不想当圣人，就别拿这个套装了，升不到最高级。

!!! info "套装特性"
    还有2封信和7本书，7本书放在背包即可提供增益。武器可以升4次，最后一次需要亡灵之路模组。

#### 精良（绿）→ 稀有（蓝）

绿升蓝的方式为触发诅咒显露条件，需要过DC，每次DC都会比前一次低。DC成功后装备上显露出诅咒，此时需要将装备取下，丢在地上，用移除诅咒法术对准装备施放，即可升级成蓝并解除诅咒。

#### 最终升级（武器）

!!! warning "最终升级注意"
    武器在升为传奇武器之后还可以升级一次，但是需要亡灵之路模组的物品。如果在亡灵之路中变为巫妖想获取所有力量，建议在打完对应的Boss后再升级武器。

找到散落在各处的7件神器，并使用套装的特性摧毁他们，当最后一个被摧毁时，你将获得最终之剑升级所需的材料。

物品在最开始拥有一个奇异封印，需要一次奥秘掷骰才能将其移除并显示实际的封印，你需要侦测善恶，然后你需要弄清楚如何移除该封印。

| 神器名称 | 获取地点 | 封印移除方法 |
| --- | --- | --- |
| 炎狱火石 | 拉斐尔在希望府邸的隐藏金库，摩尔的合同旁边 | 对其施加5次圣火 |
| 死亡位面幽影碎片 | 哀伤之邸 阴郁回廊 | 在上面连续5回合应用光耀法球 |
| 深渊心石 | 底城废墟 古代密室 | 连续5回合承受其基于最大生命值的力场伤害 |
| 受苦之心 | 巴尔神庙 奥林的房间 | 治愈它并连续5回合保持流血（注意是该物品会流血） |
| 亵渎誓约之书 | 魔法杂物店地窖 | 在拥有"永恒法典"的状态下，使用"誓约干预"技能 |
| 枢机圣匣 | 风暴海岸礼拜堂 | 摧毁其他5个神器 |
| 莫苏尔的魂器 | 枢机圣匣内 | 穿着完整的传奇套装，使用"圣光之冠净化" |

移除莫苏尔的魂器的封印后，会获得最后的两个升级材料邪孽本源和圣者灵核。

#### 联动Mod

!!! info "联动Mod"
    **亡灵之路（Path of Undeath）：**武器的最终升级所需

**觉醒的洛山达之血（Awakened Blood of Lathaner）：**背包中有洛山达之血和正义之路珍奇/传奇等级的剑，获得"燃烧的圣战"被动，近战武器攻击时造成光耀伤害，最终审判加上感知/魅力调整值，并在杀死不死生物和邪魔时造成1d10的范围溅射。传奇阶段则造成双倍调整值。如果作为吸血鬼并吸收洛山达之血然后成为"辉光不朽"（巫妖的变体），对任何东西造成相当于感知/魅力调整值两倍的光耀伤害，并在杀死不死生物和邪魔时造成1d10的范围溅射。

**觉醒的塞伦涅之矛（Awakened Spear of Selune）：**背包中有塞伦涅之矛时，激活"月华恩膏"/"神圣恩膏"时，可以移除敌人的"光耀报复"和"瞬息守护"，减慢不死生物的移动速度。如果是塞伦涅的追随者，可以增强自己和盟友。

**主职·审判官（Inquisitor）：**同样可以佩戴珍奇和传奇装备。手套解锁了审判官的圣疗。戒指使用引导誓言解锁独特的审判官动作。使用圣武士灵光的盔甲和套装效果同样也适用于审判官灵光。
