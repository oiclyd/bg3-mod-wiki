# 常见问题

## 注意事项与BUG规避

!!! warning "注意"
    以下注意事项来自大家的实测总结，遵循这些规则可以规避大部分BUG和闪退问题。

### 官方35号补丁卡加载问题

!!! tip
    官方更新35号小补丁以后，原版（无mod）也会出现卡75%和83%现象。

    解决办法：读取能进去的存档，然后进行手动存档。游玩中尽量手动存档，别开荣誉模式，可开自定义荣誉。

### 不要带亡灵召唤物跨章节

!!! tip
    跨章节请务必把亡灵类召唤物都反召唤，尤其是死灵宝可梦，否则可能卡加载。

    如果已经卡，尝试回档反召唤试试。

    队友过多的时候过章节也可能卡读档，也有水友表示遣散卡姐会卡读档，目前看来是随机事件。

    如遇到请回档 -> 反召唤 -> 保持4人队伍 -> 存读档，尝试解决。

### 坐船保持4人队伍

!!! tip
    坐船务必保持4人队伍，不然多余的队友会卡在那。

    解决办法：跑回去把岸边的队友都秒了，找耶格复活。

### 营地卡沉睡

!!! tip
    队伍超过4人时，可能遇到长休醒不来，显示有人在沉睡。
    
    点一下其他队友头像，切换到队友视角即可。

### 旅馆闪退

!!! tip
    在旅馆人数过多可能导致对话闪退，如果任何场景发生闪退请保持4人队伍。

### 频繁闪退

!!! tip
    如果你游玩频繁闪退，请把虚拟内存调高。

### 战斗闪退

!!! tip
    如果战斗闪退就把DOTA等难度暂时关掉，MCM难度调整暂时复原。

    在明拉萨入队后可能发生，但哈尔幸同时入队后就没事了。

### 将军闪退

!!! tip
    在打将军的时候记得把MCM难度调整还原。

### 巴尔刺客闪退

!!! danger
    这是拉瑞安给了空庇护术导致的，如果闪退就争取一回合秒了，秒不掉就简易作弊开秒杀。

### 自定义等级上限不要超过100级

!!! warning
    否则可能兼职没技能，等级越高技能越多越容易卡，因为被动和状态太多，不建议太高。

    一般来说 **30级足矣**，wiki内有30级经验曲线参考。

### 升级界面卡住

!!! tip
    升级或兼职如果无法抉择，请点右上角的X退出界面，再次进入升级界面即可。

    有兼职的情况卡抉择，看看前面的职业是否有东西没点选，某些等级是强制点选的。

    有些职业不能兼职，只能开卡，有些职业专属装备也是只能开卡获得。

### 剧情问题

- 无法对话是原版沉默术BUG，你被敌人沉默了，用变形术解除（比如豪华版送的变形头盔），或者再用一次沉默术。
- 威尔感叹号，长休剧情不触发也是原版BUG，是你太久没长休错过了剧情，叫你长休的时候请及时长休！
- 如果有队友在营地出不来（比如盖尔），杀掉他然后找耶格复活即可。

### 捏脸问题

- 捏脸界面有问题，或者人物显示有问题，检查游戏内设置动画层次细节是否为高。
- 魅魔仅有女性（5e规则），没做男性，所以选魅魔不要选男性。

## Mod不生效排查

1. 首先检查你的C盘Mods文件夹里面是否套娃（Mods里面还有Mods之类的情况）
2. 然后检查你的C盘Mods文件夹里面是否存在json文件，如有请删除，然后管理器排序好，启动游戏
3. 如果无效请尝试切换游戏启动项（比如本来用DX11启动，就切换另一个启动）
4. 如果无效请关闭杀毒软件，以管理员身份运行管理器
5. 如无报错：一般与mod无关

### 方法一

!!! tip
    检查管理器Settings -> Open preferences（第三栏第一个选项）

- 检查 Game Data Path 是否为：`/Steam/steamapps/common/Baldurs Gate 3/Data`
- 检查 Game Executable Path 是否为：`Steam/steamapps/common/Baldurs Gate 3/bin/bg3.exe`

### 方法二

!!! tip
    删除C盘`AppData\Local\Larian Studios\Baldur's Gate 3\PlayerProfiles\Public`里面的modsettings，然后在mod管理器里面export导出。

    如有报错：检查mod排序和冲突
