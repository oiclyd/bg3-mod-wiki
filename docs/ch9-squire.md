# 进化的波尼

**前置条件：**需要获得伊莎贝尔的祝福。


**招募方法：**


1. 去月出之塔找侍从，让她闻闻你身上的祝福。
2. 成功通过驯服检定。
3. 最后击败凯瑟里克。


**具体位置：**在你最后一次打败凯瑟里克出来的地方，靠近伊莎贝尔和艾琳重聚的地方。


**分支条件：**


- 如果伊莎贝尔还活着，且玩家同意让她加入营地，与侍从无需掷骰，告诉她你的营地，获得她的项圈。
- 如果伊莎贝尔不加入营地，或者她不在现场，玩家需要18点或以上的点数说服它加入。


**第三幕位置：**如果在第二幕结局后添加此模组，可以在第三幕以下地点找到它：


- 利文顿，神父的地下室，会睡在书架旁边的地毯上。
- 下城区，风暴海岸大礼拜堂地下室，会坐在一张桌子旁边。
