# 进化系列召唤物

进化系列召唤物是整合包中的特色系统，通过完成特定任务或收集材料，可以将原版或MOD中的召唤物进化为更强大的形态。

### 通用说明

没写的就是正常获得，就算档在第三章，大部分的召唤还是能获得，只要剧情是正常的，和角色相关的就能正常获得，要对话的不行（弗林德、伯纳德）。

同样，如果召唤物和玩家在一起时使用 **波尔** 的祝福，召唤物也会获得该祝福效果。
