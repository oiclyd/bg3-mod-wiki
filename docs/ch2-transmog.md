# 装备幻化

## 装备幻化


### MCM装备幻化说明


- 请使用 **DX11** 启动游戏，Vulkan可能不稳定或崩溃
- 创建"预设"：为整个队伍管理完整的套装，包括护甲、服装、染色、武器外观等
- 即时切换预设：在不同预设之间切换时，整个队伍的外观会立即更新
- 保留原始属性：装备在采用预设外观的同时，会保留其原始属性
- 跨槽位外观：允许在不同槽位显示物品（例如，在靴子槽显示手套）
- 附加装饰性视觉效果（VFX）：为装备和角色附加装饰性视觉效果（火焰武器、发光效果等），对游戏玩法没有影响
- 即时预览物品和染色：可以即时预览物品和染色，保存收藏夹，并隐藏不想显示的外观
- 完全支持多人游戏：联机时请保证所有人的MCM设置都一致
- 可分享的预设：可以导出为文件，发送给朋友，或者打包到你自己的模组中
- 如果使用过幻化水晶：请在MCM创建外观套装之前，先解除已装备物品的幻化


### 幻化水晶说明


- 幻化水晶是Vulkan模式下的替代品
- **请不要使用幻化水晶去幻化武器！**
